GOLD JOB AGENCY - Pakiet Landing Page
=======================================

Zawartość pakietu:
- index.html - główny plik strony internetowej
- images/ - folder z obrazkami i grafikami
- README.txt - ten plik z instrukcjami

Instrukcje uruchomienia:
1. Rozpakuj wszystkie pliki do jednego folderu
2. Otwórz plik index.html w przeglądarce internetowej
3. Strona będzie działać lokalnie bez połączenia internetowego

Opis strony:
- Kompletna landing page dla agencji pracy GOLD JOB AGENCY
- Responsywny design dostosowany do urządzeń mobilnych
- Sekcje: główna, usługi, kategorie pracy, opinie, kontakt
- Kompletne tłumaczenia polskie
- Profesjonalne style i animacje

Uwagi techniczne:
- Strona używa Tailwind CSS z CDN
- Wszystkie obrazki są lokalnie dostępne w folderze images/
- Formularz kontaktowy wymaga konfiguracji backendu dla działania
